# Wunderlandmedia Homebrew Tap

Homebrew cask formulae for [Wunderlandmedia](https://wunderlandmedia.com) macOS apps.

## HotkeyClash

Open-source macOS menu bar utility that detects keyboard shortcut conflicts across apps, config tools, and system shortcuts.

```bash
brew install --cask wunderlandmedia/tap/hotkeyclash
```

Source and issues: https://github.com/Wunderlandmedia/HotkeyClash
